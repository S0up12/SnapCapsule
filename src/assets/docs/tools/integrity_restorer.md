# Media Integrity Restorer
Scans and repairs corrupt Chat Media and Memories.

### Actions Performed:
* **Video Correction**: Fixes videos incorrectly named as .jpg.
* **Audio Extraction**: Extracts audio notes incorrectly named as .mp4.
* **Header Repair**: Repairs broken JPEG headers.

### Safety:
Originals are backed up to 'repair_backups' folders.
